# 任務卡 02 — PINBALL FEEL（彈珠手感）

> Phase 2。先讀 `Codex/00_MASTER_PROMPT.md` 與 Phase 1 成果。目標：**把「能動」變成「好玩」**，強化發射、碰撞、命中回饋，並讓 4 種釘子、3 種球全部生效。**先確保 Phase 1 仍可運行，再疊加手感。**

---

## 任務範圍（要做）

### 發射感
- 瞄準線（Line2D）顯示發射方向，可選顯示力度 / 軌跡預覽。
- 發射有明確輸入回饋（按下蓄力 / 放開發射，或點擊即射，擇一並記錄選擇）。
- 球離手有輕微的發射特效（閃光 / 粒子）。

### 碰撞回饋
- 球撞釘 / 撞牆有彈跳手感（調整彈性 `bounce`、摩擦）。
- 命中時釘子有反應（閃光、縮放彈一下、短暫變亮）。
- 命中瞬間輕微螢幕震動（screen shake，幅度克制）。

### 命中特效
- 命中爆出粒子火花（CPUParticles2D / GPUParticles2D），顏色對應該 Peg（見 `05_ART_DIRECTION.md`）。
- 球拖尾（Line2D 或粒子）。
- 善用 WorldEnvironment Glow 讓亮點發光。

### 傷害數字
- 命中時在該位置跳出傷害數字（含倍傷後的實際值），向上飄並淡出（Tween）。
- 結算時顯示「本回合總傷害」彙總。

### 音效占位（placeholder SFX）
- 發射、命中釘、落底、結算各掛一個占位音效（可用簡單嗶聲 / 內建）。
- 音效以 id / 事件驅動，方便日後替換；音量可關。

### 4 種釘子全部生效
- Normal / Burst / Heal / Double 依 `pegs.json` 的 `effect_type` 套用：
  - `damage`：累積傷害（Normal / Burst）。
  - `heal`：回復玩家 HP（Heal Peg，建議結算時統一套用）。
  - `damage_multiplier`：提升本回合倍率（Double Peg，遵守 Q-001 暫行假設：每回合 1 次、×2）。

### 3 種球全部生效
- Normal / Blast / Shield 依 `balls.json` 的 `effect_type` 套用：
  - `none`：基礎球。
  - `on_drop_bonus`：落底 / 結算時加成一次「本回合最高單次釘子傷害」（遵守 Q-002 暫行假設 A）。
  - `damage_reduction`：本回合敵人攻擊減免（遵守 Q-003 暫行假設：-30%、不疊加）。
- 註：此階段可先以「測試方式」讓 Blast / Shield 球出現以驗證效果；正式解鎖流程在 Phase 4。
